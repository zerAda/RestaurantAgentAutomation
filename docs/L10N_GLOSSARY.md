# Glossaire — EPIC5 L10N

- **Locale** : code de langue (`fr`, `ar`).
- **Script-first** : la langue de réponse est déterminée par les caractères détectés dans le message (arabe vs non).
- **Darija (translit)** : darija en alphabet latin.
- **Sticky AR** : règle optionnelle qui force AR après N messages en script arabe.
- **Template** : message paramétrable stocké en DB (`message_templates`).
- **Override tenant** : version d’un template spécifique à un tenant (ne modifie pas `_GLOBAL`).
- **Allowlist variables** : liste de variables autorisées pour un template (`message_templates.variables`).
- **RTL** : Right-to-Left (sens d’écriture arabe).
- **W4_CORE** : workflow central de traitement intents + réponse.
- **W14_ADMIN_WA_SUPPORT_CONSOLE** : console WhatsApp admin pour support + templates.
